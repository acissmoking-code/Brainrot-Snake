# Brainrot-Snake

Ein verrücktes Snake-Spiel mit Brainrot-Memes, Power-Ups und verschiedenen Modi, entwickelt mit Kivy für Python.

## Features

- **Modi**: Classic, Time Attack, Chaos, Survival
- **Power-Ups**: Speed, Slow, Ghost, Points Multiplier, Chaos
- **Brainrot-Memes**: Zufällige Statusanzeigen basierend auf "Brain Damage"
- **AdMob-Integration**: Für mobile Werbung (Android)
- **Touch-Steuerung**: Für mobile Geräte optimiert

## Installation

1. Installiere Python 3.8+
2. Installiere Kivy: `pip install kivy`
3. Für Android: Build mit Buildozer oder ähnlich

## Ausführen

```bash
python main_Version6.py
```

## Steuerung

- Pfeiltasten oder W/A/S/D für Desktop
- Touch-Gesten für Mobile

## Modi

- **Classic**: Standard Snake-Spiel
- **Time Attack**: 60 Sekunden Zeitlimit
- **Chaos**: Häufige Power-Ups und zufällige Richtungen
- **Survival**: Wände und Feinde
